# network-troubleshooting-lab
Basic network troubleshooting lab using Windows CMD tools (ping, ipconfig, nslookup, tracert).


Overview:
This project demonstrates basic network troubleshooting using Windows command-line tools.

Tools Used:
- ipconfig
- ping
- nslookup
- tracert

Skills Demonstrated:
- IP configuration analysis
- DNS troubleshooting
- Network connectivity testing
- Routing path analysis
